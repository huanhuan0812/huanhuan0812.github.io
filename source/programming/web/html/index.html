HTML 基础 ​

HTML（HyperText Markup Language）是用于创建网页的标准标记语言。下面是HTML的基础知识：

基本结构 ​

html

<html>
<head>
    <title>页面标题</title>
</head>
<body>
    <h1>我的第一个标题</h1>
    <p>我的第一个段落。</p>
</body>
</html>

常用属性 ​

  • class - 为元素指定类名
  • id - 为元素指定唯一ID
  • style - 指定内联CSS样式
  • title - 指定元素的额外信息（显示为工具提示）
